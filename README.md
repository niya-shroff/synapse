# Synapse
An OpenSource MentraOS Smart Glasses App - Connections between people + memory

The goal is to develop a "Social Intelligence Layer" with real-time people context.
* Recognizes people (opt-in)
* Shows:
    * Name
    * Where you met
    * Last convo summary
    * Suggested talking points
* Uses camera + memory + AI
* Turns networking into a superpower
* Taking facial recognition to the next level for keep track of people you have met
